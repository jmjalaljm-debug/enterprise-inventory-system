# enterprise-inventory-system
نظام احترافي لإدارة مستودع أكواب بلاستيكية مطور بـ PHP، MySQL، وTailwind CSS. يوفر لوحة تحكم تفاعلية للمؤشرات الحيوية وإدارة المنتجات، وتتبع دقيق لحركات الوارد والصادر، مع رسوم بيانية مباشرة وبحث فوري. يتميز بتصميم عصري (Dark Theme &amp; Glassmorphism) يضمن تجربة مستخدم سريعة وقابلة للتطوير.
